# Privacy Policy

This Privacy Policy describes how your information is handled when you use the **Android Dependency Resolver** plugin ("Software").

### 1. Information Collection
The Software **does not collect, store, or transmit any personal information**.

### 2. Data Usage
The Software works locally within your IDE. It communicates with a remote Cloudflare API solely to fetch the latest library metadata (versions, coordinates, and compatibility rules). During this request:
- No project data, source code, or personal identifiers are sent to the server.
- Only generic HTTP request headers (such as User-Agent) typical of IDE communication are transmitted.

### 3. Local Data
All project analysis (Kotlin version, AGP version, Gradle configuration) is performed locally on your machine. This data is used only to filter the library suggestions within the IDE tool window and is never uploaded anywhere.

### 4. Third-Party Services
The Software uses a custom Cloudflare Pages API to fetch library definitions. Use of this service is subject to Cloudflare's privacy policy.

### 5. Changes to This Policy
We may update our Privacy Policy from time to time. You are advised to review this page periodically for any changes.
